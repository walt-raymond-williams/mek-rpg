# Injuries Recovery
## Purpose
What this rule is for.
## When to Use
Use this rule when injuries affect downtime, missions, or long-term play.
## Do Not Use For
Cases where another subsystem should be used.
## Basic Procedure
1. Review current wounds and treatment access.
2. Apply verified recovery timing and consequences.
3. Update mission readiness and campaign state.
## Practical GM Guidance
Use injury recovery to create meaningful downtime choices without stalling play.
## Common Edge Cases
- Edge case: Severe injuries may require specialist care.
- Edge case: Kid-friendly mode changes narration and lethal consequence handling.
