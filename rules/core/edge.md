# Edge

## Purpose

Edge represents a limited luck or dramatic intervention resource. A character can burn Edge to influence an Attribute Check or Skill Check that directly affects that character, either by changing the result or forcing a reroll.

## When to Use

- A player wants to improve their own important roll.
- A player wants to worsen or reroll a roll that directly affects their character, such as an attack against them.
- The GM allows an important NPC to use Edge in a high-stakes scene.
- A character needs to recover previously burned Edge.

## Do Not Use For

- Helping another character's unrelated roll.
- Repeatedly modifying or rerolling the same check.
- Bypassing a rule subsystem that should be resolved normally.
- Routine NPC rolls, unless the NPC is important enough that the extra handling is worth the table time.

## Basic Procedure

1. Confirm the roll is an Attribute Check or Skill Check and directly affects the character spending Edge.
2. Confirm the character has unburned Edge remaining.
3. Spend Edge no more than once for that roll.
4. If spending before the roll, apply a larger modifier based on the Edge spent; it can help the character's roll or hinder an opposing roll.
5. If spending after the roll, either apply a smaller result adjustment or force a reroll, depending on the player's choice.
6. Reduce the character's available Edge by the points burned.
7. Recover burned Edge only through GM-approved recovery, XP buyback, or the source's zero-Edge recovery requirement.

## Practical GM Guidance

Ask for the Edge decision at the correct timing: before the roll for a bigger but riskier modifier, after the roll for a more controlled but weaker adjustment or a reroll. Track burned Edge visibly because recovery is not automatic in ordinary scene time.

For NPCs, use Edge sparingly. Routine opposition should not spend it because it slows play and can feel arbitrary. Reserve it for named rivals, major allies, and consequential threats.

## Common Edge Cases

- Edge is not generally a normal linked Attribute for Skills.
- A character cannot spend Edge for someone else's unrelated check.
- A forced reroll replaces the previous roll rather than stacking earlier Edge modifiers onto the new result.
- Natural fumbles and high-roll results can interact with Edge timing; verify the exact sequence from the cited pages when it matters.
- Characters reduced to zero Edge have a special recovery constraint in the source.
