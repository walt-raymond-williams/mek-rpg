# Skill Checks

## Purpose

Skill Checks resolve actions where a character has relevant training. They combine the Skill's target number, the character's Skill level, linked Attribute modifiers, specialization, and situational modifiers.

## When to Use

- The character is attempting an action covered by a Skill they possess.
- The task depends on training, technique, professional knowledge, or practiced combat ability.
- The character sheet lists a Skill target number and complexity for the action.

## Do Not Use For

- Attempts by characters with no relevant Skill, including no level 0; use an Attribute Check instead.
- Tasks where the GM decides no roll is needed.
- Opposed contests without reading `rules/core/opposed-actions.md`.
- Tactical BattleTech combat once the scene depends on tabletop unit rules.

## Basic Procedure

1. Identify the applicable Skill and confirm the character has it.
2. Use the Skill's listed target number and complexity.
3. Roll 2D6.
4. Add the character's Skill level.
5. Add any linked Attribute modifier, specialization modifier, and situational modifier that applies.
6. Compare the final result to the target number. If degree matters, use margin guidance in `rules/core/basic-action-resolution.md`.

## Practical GM Guidance

Choose the narrowest Skill that actually fits the action. A broad plan may require only one check if the interesting uncertainty is the first obstacle; avoid chaining rolls unless the character's goal clearly enters a new phase with new consequences.

## Common Edge Cases

- A Skill can sometimes solve only part of a goal. For example, establishing a technical connection may not also retrieve, decrypt, or interpret everything behind it.
- Specialization can help or hinder depending on whether the specialty matches the immediate task.
- If the character lacks the Skill, do not use the Skill target number; convert the attempt to an Attribute Check using the Skill's linked Attributes.
- Combat Skills may route into personal combat summaries before the roll is resolved.
