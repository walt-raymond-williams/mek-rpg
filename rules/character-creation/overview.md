# Character Creation Overview
## Purpose
What this rule is for.
## When to Use
Use this rule when creating or reviewing a player character.
## Do Not Use For
Cases where another subsystem should be used.
## Basic Procedure
1. Establish character concept and campaign fit.
2. Build attributes, lifepaths, traits, and skills using verified source summaries.
3. Record advancement hooks and campaign connections.
## Practical GM Guidance
Tie creation choices to playable contacts, obligations, and mission roles.
## Common Edge Cases
- Edge case: A tactical pilot may also need Classic BattleTech pilot notes.
- Edge case: Unverified character options should be marked for source review.
