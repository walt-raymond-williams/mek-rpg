# Melee Attacks
## Purpose
What this rule is for.
## When to Use
Use this rule when characters attack at close personal range.
## Do Not Use For
Cases where another subsystem should be used.
## Basic Procedure
1. Establish reach, position, weapon, and intent.
2. Apply verified melee attack procedure.
3. Resolve damage, control, movement, or other consequences.
## Practical GM Guidance
Use melee to create movement, disarms, grapples, and immediate danger, not only damage.
## Common Edge Cases
- Edge case: Grappling or special maneuvers may need source review.
- Edge case: Mech-scale physical attacks belong in tactical BattleTech.
