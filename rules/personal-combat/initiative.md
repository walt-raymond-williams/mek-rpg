# Initiative
## Purpose
What this rule is for.
## When to Use
Use this rule when the order of action matters during personal combat or tense scenes.
## Do Not Use For
Cases where another subsystem should be used.
## Basic Procedure
1. Identify all acting sides or characters.
2. Apply the verified initiative procedure.
3. Resolve actions in order until the scene changes.
## Practical GM Guidance
Use side initiative or grouped NPC action if it keeps play moving and matches the source.
## Common Edge Cases
- Edge case: Surprise or ambush may alter the first exchange.
- Edge case: Tactical BattleTech has its own initiative handling.
