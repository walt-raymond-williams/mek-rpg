# Personal Combat Overview
## Purpose
What this rule is for.
## When to Use
Use this rule when characters fight at personal scale rather than BattleMech tactical scale.
## Do Not Use For
Cases where another subsystem should be used.
## Basic Procedure
1. Establish participants, intent, and immediate danger.
2. Resolve initiative, attacks, damage, and consequences using verified summaries.
3. End combat when surrender, escape, incapacitation, or tactical handoff occurs.
## Practical GM Guidance
Keep personal combat fast and focused on objectives, cover, and consequences.
## Common Edge Cases
- Edge case: BattleMech combat should switch to Classic BattleTech.
- Edge case: Social or stealth conflict may avoid combat entirely.
