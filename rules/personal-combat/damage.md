# Damage
## Purpose
What this rule is for.
## When to Use
Use this rule when an attack, hazard, or accident harms a character.
## Do Not Use For
Cases where another subsystem should be used.
## Basic Procedure
1. Determine the source and amount of harm using verified rules.
2. Apply armor, resistance, or mitigation if applicable.
3. Record wounds, conditions, or other consequences.
## Practical GM Guidance
Describe consequences clearly without over-detail. For kid-friendly mode, avoid graphic narration.
## Common Edge Cases
- Edge case: Vehicle and BattleMech damage should use tactical rules.
- Edge case: Nonlethal harm needs source review.
