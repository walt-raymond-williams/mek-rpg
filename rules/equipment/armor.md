# Armor
## Purpose
What this rule is for.
## When to Use
Use this rule when personal armor changes injury risk or combat outcomes.
## Do Not Use For
Cases where another subsystem should be used.
## Basic Procedure
1. Identify armor type and verified protection rules.
2. Apply armor during damage resolution.
3. Track armor limits, damage, or encumbrance if applicable.
## Practical GM Guidance
Make armor useful but not invisible. It can affect stealth, heat, authority, or social access.
## Common Edge Cases
- Edge case: BattleMech armor uses tactical rules.
- Edge case: Called shots or armor gaps need source review.
