# Core Resolution Overview
## Purpose
Summarize how uncertain actions are resolved at the table.
## When to Use
Use this rule when a character attempts an action with uncertain outcome and meaningful consequences.
## Do Not Use For
Cases where personal combat, character creation, or tactical BattleTech procedures should be used.
## Basic Procedure
1. Identify the intended action and stakes.
2. Choose the relevant check procedure and modifiers.
3. Resolve the result and apply consequences.
## Practical GM Guidance
State the risk before calling for a roll and avoid rolling when success or failure is obvious.
## Common Edge Cases
- Edge case: The action may be better handled as an opposed check.
- Edge case: The scene may need tactical combat instead of a single check.
