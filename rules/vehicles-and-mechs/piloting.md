# Piloting
## Purpose
What this rule is for.
## When to Use
Use this rule when a character controls a vehicle or BattleMech under risky conditions.
## Do Not Use For
Cases where another subsystem should be used.
## Basic Procedure
1. Identify vehicle type, situation, and stakes.
2. Apply verified piloting check procedure.
3. Resolve movement, control, mishap, or tactical handoff.
## Practical GM Guidance
Use piloting checks for meaningful danger: bad terrain, evasive movement, damage, or time pressure.
## Common Edge Cases
- Edge case: Tactical BattleTech has specific piloting rules.
- Edge case: Routine travel should not require repeated rolls.
