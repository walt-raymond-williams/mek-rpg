# Vehicles and BattleMechs Overview
## Purpose
What this rule is for.
## When to Use
Use this rule when RPG characters operate vehicles, BattleMechs, or other tactical assets.
## Do Not Use For
Cases where another subsystem should be used.
## Basic Procedure
1. Determine whether the scene stays in RPG mode or switches to tactical combat.
2. Use verified piloting, gunnery, and MechWarrior skill summaries.
3. Record any tactical consequences in campaign state.
## Practical GM Guidance
Keep RPG-scale vehicle moments simple unless tactical detail is the point of the scene.
## Common Edge Cases
- Edge case: Full BattleMech combat should hand off to Classic BattleTech.
- Edge case: Pilot conversion needs verified source and table policy.
