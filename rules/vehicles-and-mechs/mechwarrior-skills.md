# MechWarrior Skills
## Purpose
What this rule is for.
## When to Use
Use this rule when an RPG character's BattleMech or vehicle competence matters.
## Do Not Use For
Cases where another subsystem should be used.
## Basic Procedure
1. Identify the relevant piloting, gunnery, or support skill.
2. Apply the verified RPG procedure or conversion note.
3. Record tactical ratings if needed for BattleTech handoff.
## Practical GM Guidance
Keep separate notes for RPG skills and tactical pilot/gunnery values until conversion is verified.
## Common Edge Cases
- Edge case: Converting to Classic BattleTech requires a table policy.
- Edge case: Damaged units may impose tactical constraints.
