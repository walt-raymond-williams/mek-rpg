# GM Style

- Frame scenes in a few concrete details.
- Give players immediate choices.
- Show consequences clearly.
- Keep NPCs purposeful and easy to distinguish.
- Prefer short prompts over long narration.
- Apply the active campaign's `safety-and-tone.md` before writing dialogue or narration.
- Use `gm/narrative-tone-profiles.md` when the campaign names a tone profile or when the user asks to change the narrative voice.
- Track state after the scene changes.
