# Touring The Stars

## Purpose

Use this routing aid when the GM needs broad worldbuilding categories for society, culture, government, law, education, work, media, transportation, religion, economics, industry, or socio-industrial level.

## When to Use

- A mission moves to a new world and the GM needs a quick local profile.
- The table asks how ordinary life works on a planet.
- A scene depends on government type, law enforcement, legal exposure, education, labor, communications, travel, religion, industry, or import dependence.
- The GM needs to interpret or create a Universal Socio-Industrial Level Reference code.

## Do Not Use For

- Reproducing the Universal Socio-Industrial Level Reference table or category text.
- Copying setting essays about societies, cultures, governments, economies, industries, or technologies.
- Treating broad worldbuilding guidance as exact canon for a named planet.
- Real-world legal, economic, educational, or technical advice.

## Basic Procedure

1. Decide which world layer matters to the scene:
   - government and law
   - policing and courts
   - education and class access
   - work and local economy
   - communications and media
   - transport and infrastructure
   - religion or public values
   - technology and industry
   - import dependence and resources
2. If a USILR code is available, use it as a prompt for local technology, industry, resources, output, and agriculture/population pressure. Inspect the cited source for exact code meanings.
3. Translate the layer into one practical obstacle, one opportunity, and one NPC-facing detail.
4. Keep faction-wide assumptions flexible. Local planets can diverge from their realm's broad pattern.
5. Record durable world facts in the active campaign save.

## Practical GM Guidance

- Use worldbuilding categories to answer "what changes the PCs' choices here?" rather than building an encyclopedia entry.
- Government type matters most when it changes permits, courts, military response, corruption, public speech, travel, or property.
- Education and work patterns are useful for NPC competence, class tension, labor disputes, recruitment, and local knowledge.
- Communications and transport shape time pressure: what can be known, how fast help arrives, and who can leave.
- Industry and imports shape mission stakes: spare parts, medical access, food, fuel, repairs, and salvage value.
