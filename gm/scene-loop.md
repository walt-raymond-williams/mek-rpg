# Scene Loop

1. Establish location, stakes, and immediate pressure.
2. Ask what the player does.
3. If the action is safe or obvious, resolve it without a roll.
4. If failure matters, read `indexes/task-router.md`, then the routed summary, before calling for the most relevant roll and stating the risk.
5. Apply consequences, update state, and present the next decision.

Use 2-4 options when the player seems unsure. Options should be concrete actions, not broad advice.

End a scene when the objective is resolved, the situation changes location, or a tactical encounter begins.
