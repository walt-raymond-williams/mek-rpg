## Selection Rules

- Load exactly one campaign folder for normal play.
- Do not mix playtest-only facts into table canon unless the user explicitly promotes them.
- If the user asks to resume play and no active campaign is selected, ask which campaign to load.
- Shared rules, indexes, and GM procedures are global. PCs, NPCs, assets, locations, relationships, missions, hooks, and session notes belong inside the active campaign folder.
