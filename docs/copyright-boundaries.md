# Copyright Boundaries

This project must avoid redistributing copyrighted rulebook content.

Allowed committed content:

- Original campaign notes.
- Paraphrased rule summaries.
- Procedures and examples written in original language.
- Page references to a legally owned source.
- Routing indexes and manifest metadata.

Not allowed in committed content:

- Purchased PDFs or EPUBs.
- Raw extracted source text.
- Large verbatim passages from the rulebook.
- Tables copied from the rulebook unless explicitly licensed for reuse.

When uncertain, keep the detail in a private ignored source file and write only a paraphrased summary with page references.
