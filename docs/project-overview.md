# Project Overview

MEK RPG combines a private rules index with a practical GM workspace. The project stores paraphrased summaries and references to a legally owned A Time of War source, then uses those summaries to answer rules questions and run scenes.

The current project profile and mode router live in `docs/current/MEK_RPG_PROJECT_PROFILE.md`.

Primary modes:

- GM mode: run scenes, call for meaningful rolls, update campaign state, and keep play moving.
- Rules lookup: route a question to the right summary, answer from project files, and cite page references when needed.
- Project development: improve docs, scripts, indexes, summaries, issues, handoffs, and workflow state.
- Tactical handoff: move to Classic BattleTech, MegaMek, or MekHQ when positioning, armor, heat, initiative, and weapon ranges matter.
- Source processing: explicit-request-only extraction of private page text, followed by paraphrased summaries and indexes.
