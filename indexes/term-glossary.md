# Term Glossary

Status: source-reviewed for issue `#46` where source pages are listed. This is a routing aid, not a replacement for rules summaries or the owned source.

Use this file when a user uses a BattleTech, A Time of War, or workspace shorthand term and the assistant needs to identify the right summary, GM procedure, or ownership boundary.
